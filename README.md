Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @codesnooker/number-formatter`

## Usage

    var numFormatter = require('@codesnooker/number-formatter');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`
